## JavaScript Variables ‚Äì Quick Reference

### What is a Variable?

A variable is a **named container** used to store data that can be used or changed later.

---

### Declaring Variables

| Keyword | Use case (Very simple)      |
| ------- | --------------------------- |
| `var`   | Old way (avoid in new code) |
| `let`   | Value can change            |
| `const` | Value should NOT change     |

```javascript
let age = 18;
const siteName = "MyWebsite";
```

---

### Rules for Variable Names

‚úî Must start with a letter, `_` or `$`
‚úî Can contain numbers
‚ùå Cannot start with a number
‚ùå No spaces

```javascript
let userName = "Rahul";   // valid
let _count = 10;         // valid
```

## JavaScript Data Types ‚Äì Quick View

JavaScript is **dynamically typed** (no need to specify type).

---

### 1. Number

Used for **integers and decimals**

```javascript
let marks = 85;
let price = 99.50;
```

---

### 2. String

Used for **text**

```javascript
let name = "Anita";
let city = 'Pune';
```

---

### 3. Boolean

Used for **true / false**

```javascript
let isLoggedIn = true;
let isPaid = false;
```

---

### 4. Undefined

Declared but **no value assigned**

```javascript
let result;
```

---

### 5. Null

Represents **empty or no value intentionally**

```javascript
let data = null;
```

---

### 6. Object

Used to store **multiple values together**

```javascript
let student = {
  name: "Amit",
  age: 20
};
```

---

### 7. Array

Used to store **list of values**

```javascript
let subjects = ["Maths", "Science", "English"];
```

---

### 8. typeof Operator

Used to **check data type**

```javascript
typeof 10        // "number"
typeof "Hi"      // "string"
typeof true      // "boolean"
```

---

## Very Important Notes (Exam + Interview)

‚úî JavaScript runs **inside the browser**
‚úî Variables can change type anytime
‚úî Use `let` and `const` in modern JS
‚úî Arrays and Objects are special types of objects
